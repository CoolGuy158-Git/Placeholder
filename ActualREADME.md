## Is there any use to it?
Well actually, YES! You see, if you want to learn how to contribute this is actually a bit useful.

# Contributing
## Making/Updating stuff
Yes you can update stuff
You can make your own mini project (It can be any language) just add it to the Code folders.
But only use Variable/Function names that are witten on the UsableNames.txt
Each project file should include this header:
```python
"""
Name
License
----
Author
"""
```
## Bug report
If you find any bugs please report them while citing:
- What you expected
- What happened
- How to replicate
- And screenshots (If available)
